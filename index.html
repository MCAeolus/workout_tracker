Workout Tracker

Track your progress, one rep at a time

Setup Required

Step 1: Create Google Sheets Template

Create a new Google Sheet with the following structure:

Sheet 1: "Routines"

Columns: RoutineName, ExerciseName, Notes, DefaultSets

Sheet 2: "WorkoutLog"

Columns: Date, Routine, Exercise, SetNumber, Weight, Reps, Notes

Step 2: Create OAuth 2.0 Credentials

1. Go to Google Cloud Console

2. Create a new project or select existing

3. Enable "Google Sheets API"

4. Go to "APIs & Services" → "Credentials"

5. Click "Create Credentials" → "OAuth client ID"

6. If prompted, configure the OAuth consent screen:

- User type: External

- Add your email as a test user

- Scopes: Just use default (no need to add specific scopes)

7. Application type: "Web application"

8. Add Authorized JavaScript origins: https://[your-username].github.io

9. Copy your Client ID (looks like: 123456-abc.apps.googleusercontent.com)
